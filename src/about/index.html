Why?

MeOS is the successor of WebOS. WebOS was a project that I started for fun. The goal was to create a website that looked like an operating system. The challenge was to code it without using any libraries, frameworks, or external resources (except for the icons). WebOS was a fun project, but the larger it grew, the more spaghetti code it became. So I decided to start from scratch.

MeOS is the result of that. But now, the goal is a different one. Like before, the goal is to create a website that looks like an operating system.

Tech Stack

As described above, MeOS like WebOS, is built without any libraries, frameworks, or external resources (except for the icons).
