Why?

For fun and to challenge myself. I wanted to create a website that looks like an operating system without using any help from libraries, frameworks, or external resources. It was also a good opportunity to learn why you should use libraries and frameworks in the first place.

Tech Stack

MeOS is built without any libraries, frameworks, or external resources (except for the icons).
